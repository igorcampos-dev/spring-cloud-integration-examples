# Spring AWS SQS Example

A simple project demonstrating the integration of **Spring Boot** with **Amazon SQS**.

It includes a basic **SQS Producer** and **Consumer** to send and receive messages. The project uses **LocalStack** for local AWS emulation and supports Docker for easy setup.

The purpose of this project is to provide a straightforward example of how to work with Spring and Amazon SQS in a practical scenario.
